Write a java program to print "Hello Java!"


Published on Jan 28, 2024
Rajesh Bhola

Introduction

Welcome to bhola{code}camp, where you can build your logical problem solving skills by practicing the code programs.


In this article, I am just writing a simple hello java program in java. I will also explain the syntax that it has used so that you can remember at its core level. Without delay lets get started...


Code:

    
        public class Test {
            public static void main(String[] args) {
                System.out.println("Hello Java!");
            }
        }

public class Test: This declares a class named Test. In Java, every application must contain at least one class with a main method.


public static void main(String[] args): This is the main method. It is the entry point of the program. When you run a Java application, the Java Virtual Machine (JVM) looks for the main method and starts executing the code from there.

The main method is declared as public, which means it can be accessed from outside the class. It is also declared as static, indicating that it belongs to the class rather than an instance of the class. The void keyword indicates that the main method does not return any value.


System.out.println("Hello Java!");: This line prints the string "Hello Java!" to the console. The System.out is an instance of the PrintStream class that provides methods to output data. The println method is used to print the specified string followed by a newline character, so each call to println results in a new line in the console.


Output:

    
      Hello Java!
